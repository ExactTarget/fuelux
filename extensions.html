Submit an extension

To submit your extension to the Fuel UX Community, publish a repository on GitHub with complete and well-written instructions. Instructions should include a clear and detailed explanation of the purpose and usage of the extension. Please show clear and easy-to-read code for display and copying, when applicable. Consider including screenshots in your README.md. If you choose not to support your extension, state this in your description.

Please fork the Fuel UX repository and submit a pull request with a modified extensions.html in the gh-pages branch.

Fuel UX controls are represent jQuery plugins, so you can $.extend() them, too.
