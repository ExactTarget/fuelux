Spinbox spinbox.js

Example

Spinbox includes an enhanced numeric input based upon the native <input type="number">. Wrap the input and the increment/decrement buttons within .fuelux .spinbox

Sample Methods

Usage

A spinbox allows for click interaction with a numeric input.

Via data-attributes

The spinbox will automatically instantiate any spinbox with data-initialize="spinbox". If you add control markup after page load with data-initialize="selectlist", this control initializes on mousedown.

Via JavaScript

Call the selectlist control via JavaScript:

{% highlight js %}$('#MySpinbox').spinbox();{% endhighlight %}

Markup

Add class="spinbox" to an a div within a class="fuelux" container.

Methods

.spinbox('destroy')

Removes all functionality, jQuery data, and the markup from the DOM. Returns string of control markup.

{% highlight js %}$('#element').spinbox('destroy');{% endhighlight %}

.spinbox('value')

Sets or returns the spinner value, as shown in the example below:

{% highlight js %}$('#element').combobox('value') {% endhighlight %}

This example would return the current value of the spinbox and

{% highlight js %}$('#element').combobox('value', 37) {% endhighlight %}

would set the current value to 37. If value contains units, the value returns as a string. Otherwise, the value returns as a number.

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-selected="".

Name type default description
value number 1 Sets the initial spinner value
min number 1 Sets the minimum spinner value
max number 999 Sets the maximum spinner value
step number 1 Sets the increment by which the value in the spinner will change each time the spinner buttons are pressed
hold boolean true If true, the spinner will react to its buttons being pressed and held down
hold boolean true If true, the spinner will react to its buttons being pressed and held down
speed string "medium" Assigns spinner speed when buttons are held down. Options include "fast","medium","slow".
disabled boolean false Creates a disables spinner.
units array [] Units that will be allowed to appear and be typed into the spinbox input along with the numeric value. For example, setting units to a value of ['px', 'en', 'xx'] would allow px, en, and xx units to appear within the spinbox input

Events

Event Type Description
changed.fu.spinbox This event fires when the value changes (either by selecting an item from the list or updating the input value directly). Arguments include event, value where value is the current value of the spinner. Returns the current value of the spinner.
{% highlight js %} $('#MySpinbox').on('changed', function () { // do something }) {% endhighlight %}
