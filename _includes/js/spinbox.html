Spinbox spinbox.js

Example

Spinbox is an enhanced numeric input based upon the native <input type="number">. Wrap the input and the increment/decrement buttons within .fuelux .spinbox

Usage

A spinbox allows for click interaction with a numeric input.

Trigger the Spinbox via JavaScript:

{% highlight js %}$('#MySpinbox').spinbox();{% endhighlight %}

Markup

Add class="spinbox" to an a div within a class="fuelux" container.

Methods

.spinbox('value')

Sets or returns the spinner value. As an example:

{% highlight js %}$('#element').combobox('value') {% endhighlight %}

would return the current value of the spinbox and

{% highlight js %}$('#element').combobox('value', 37) {% endhighlight %}

would set the current value to 37. If value contains units it will be returned as a string, otherwise it will be returned as a number

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-selected="".

Name type default description
value number 1 Sets the initial spinner value.
min number 1 Sets the minimum spinner value.
max number 999 Sets the maximum spinner value.
step number 1 Sets the increment by which the value in the spinner will change each time the spinner buttons are pressed.
hold boolean true If true the spinner will react to its buttons being pressed and held down.
hold boolean true If true the spinner will react to its buttons being pressed and held down.
speed string "medium" Spinner speed when buttons are held down. The options include: "fast","medium","slow".
disabled boolean false Spinner will be disabled when it is created.
units array [] Units that will be allowed to appear and be typed into the spinbox input along with the numeric value. For example, setting units to a value of ['px', 'en', 'xx'] would allow px, en, and xx units to appear within the spinbox input

Events

Event Type Description
changed This event is fired when the value has changed (either by selecting an item from the list or updating the input value directly). Arguments are event, value where value is the current value of the spinner. Current value of the spinner is returned.
{% highlight js %} $('#MySpinbox').on('changed', function () { // do something }) {% endhighlight %}
