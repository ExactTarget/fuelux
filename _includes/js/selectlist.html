Usage

This is a single select drop-down similar to the native select control. Data attributes are added to the li tag.

Via data-attributes

To enable the selectlist control without writing Javascript, add data-initialize="selectlist" to the .selectlist element that you wish to initialize. Such elements that exist when $.ready() executes will be initialized. Any combobox markup that is programatically created with data-initialize="selectlist" sometime after the initial page load will not immediately initialize. Rather, it will initialize when the mousedown event is fired on it.

Via JavaScript

Call the selectlist control via JavaScript:

{% highlight js %}$('#mySelectlist').selectlist();{% endhighlight %}

Methods

$().selectlist()

Attaches selectlist to an input and a button group

.seleclist('destroy')

Removes all functionality, jQuery data, and the markup from the DOM. Returns string of control markup.

{% highlight js %}$('#mySelectlist').selectlist('destroy');{% endhighlight %}

.selectlist('disable')

Disables the component

{% highlight js %}$('#mySelectlist').selectlist('disable');{% endhighlight %}

.selectlist('enable')

Enables the component

{% highlight js %}$('#mySelectlist').selectlist('enable');{% endhighlight %}

.selectlist('selectedItem')

Returns an object containing the jQuery data() contents of the selected item. This data includes .data-* attributes plus a text property with the visible text from the item.

{% highlight js %}$('#mySelectlist').selectlist('selectedItem');{% endhighlight %}

.selectlist('selectByIndex')

Set the selected item based on its index in the list (zero-based index). Convenience method for .selectBySelector('li:eq({index})')

{% highlight js %}$('#mySelectlist').selectlist('selectByIndex', '1');{% endhighlight %}

.selectlist('selectByText')

Set the selected item based on its text value

{% highlight js %}$('#mySelectlist').('selectByText', 'Four');{% endhighlight %}

.selectlist('selectBySelector')

Set the selected item based on a selector

{% highlight js %}$('#element').('mySelectlist', 'li[data-fizz=buzz]');{% endhighlight %}

.selectlist('selectByValue')

Set the selected item based on its value. Convenience method for .selectBySelector('data-value={value}') that requires the item to include a .data-value="{value}" attribute

{% highlight js %}$('#mySelectlist').('selectByValue', '1');{% endhighlight %}

Events

Event Type Description
changed.fu.selectlist This event fires when the value changes by selecting an item from the list. Arguments include event, data where data represents the same object structure returned by the selectedItem method.
{% highlight js %} $('#mySelectlist').on('changed.fu.selectlist', function () { // do something }); {% endhighlight %}
