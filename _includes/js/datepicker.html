Datepickers [STUB] datepicker.js

Examples

Choose a data from a calendar drop-down or use moment.js to format and parse various date formats.

To support various date formats around the world, datepicker includes an optional dependency on moment.js. Download moment.js (with languages).

Usage

Via data-attributes

The datepicker will automatically instantiate any datepickers with data-initialize="datepicker". If you add the control markup after page load with data-initialize="datepicker", this control initialize on mousedown.

Via JavaScript

Call the datepicker control via JavaScript:

{% highlight js %}$('#MyDatepicker').datepicker();{% endhighlight %}

Markup

Add class="datepicker" to an input and a drop-down within a class="fuelux" container.

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-selected="".

Name type default description
date date new Date() If date is set to null when the datepicker is initialized, the getDate method will return null until the user or a program selects a date. (programmatically or by the user). Otherwise, the code will parse the date.
createInput object false Allows control to generate HTML Markup. You can set this value to true or false or use an object like the example below: { native: false, dropDownBtn: false, inputSize: 'span3' } native can include a value of true or false. If true, the code will ignore dropDownBtn . dropDownBtn can include a value of true or false. inputSize can either include a CSS class or a integer.
dropdownWidth integer 170 Sets pixel width of drop-down
restrictDateSelection boolean true Disables ability to select dates in the past. Based on the current date.
parseDate date return new Date( date ); Function allows custom date parsing and passes the date as a parameter. This function returns a Date Object. Do not overwrite this function if using Moment.js.
formatDate string MM-DD-YYYY Function allows custom formatting of date, but will never modify the main Date Object. Passes the date as a parameter. This function returns a formatted Date Object. Do not overwrite this function if using Moment.js.
blackoutDates boolean true Function restricts date selection. Passes the date as a parameter. Boolean value should return false
momentConfig Object true Handles culture and format if Moment.js is available { culture: 'en', formatCode: 'L' }

Methods

$().datepicker(options)

Attaches a datepicker to an input and a drop-down menu

.datepicker('destroy')

Removes all functionality, jQuery data, and the markup from the DOM. Returns string of control markup.

{% highlight js %}$('#element').datepicker('destroy');{% endhighlight %}

.datepicker('disable')

Disables the component

{% highlight js %}$('#element').datepicker('disable'){% endhighlight %}

.datepicker('enable')

Enables the component

{% highlight js %}$('#element').datepicker('enable'){% endhighlight %}

.datepicker('getDate')

Returns the selected date (not formatted)

{% highlight js %}$('#element').datepicker('getDate'){% endhighlight %}

.datepicker('getFormattedDate')

Returns the selected formatted date

{% highlight js %}$('#element').datepicker('getFormattedDate'){% endhighlight %}

.datepicker('setDate')

Sets datepicker to date provided. Date provided must use a valid Date Object

{% highlight js %}$('#element').datepicker('setDate', new Date() ){% endhighlight %}

.datepicker('getCulture')

Returns the culture currently used by the datepicker. Only available using Moment.js with Langs

{% highlight js %}$('#element').datepicker('getCulture'){% endhighlight %}

.datepicker('setCulture')

Updates the culture used by the datepicker. Only available using Moment.js with Langs

{% highlight js %}$('#element').datepicker('setCulture'){% endhighlight %}

.datepicker('getFormatCode')

Returns the format code currently used by the datepicker. Only available using Moment.js with Langs

{% highlight js %}$('#element').datepicker('getFormatCode'){% endhighlight %}

.datepicker('setFormatCode')

Updates the format code used by the datepicker. Only available using Moment.js with Langs

{% highlight js %}$('#element').datepicker('setFormatCode'){% endhighlight %}
