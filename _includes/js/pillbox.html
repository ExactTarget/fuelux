Usage

A pillbox allows user tag management and enables features such as typahead and tag editing.

Via data-attributes

The pillbox will automatically instantiate any pillboxes with data-initialize="pillbox". If you add the control markup after page load with data-initialize="pillbox", this control will intialize on mousedown.

Via JavaScript

Call the pillbox via JavaScript:

{% highlight js %}$('#MyPillbox').pillbox();{% endhighlight %}

Markup

Add class="pillbox" to a div within a class="fuelux" container.

{% highlight html %}
  • Item Title Remove
    • ....
    {% endhighlight %}

    Methods

    .pillbox('addItems')

    Adds Items to the pillbox programatically

    {% highlight js %} $('#element').pillbox('addItems', index, [{ text: '', value: '' }] ); {% endhighlight %}
    Parameter Description
    index Required. Indicates the position to start inserting pill(s). The first pane exists at position 1. If the value equals -1, the item will append at the end of the list of pills.
    [item1, ..., itemX]
    item1, ..., itemX    		 An array or list of pill objects to be added at the index. Each pill can contain a label text and value.

    .pillbox('destroy')

    Removes all functionality, jQuery data, and the markup from the DOM. Returns string of control markup.

    {% highlight js %}$('#element').pillbox('destroy');{% endhighlight %}

    .pillbox('removeItems')

    Remove items from the pillbox programatically. The first parameter includes a 1 based index that represents the location of the first element to be removed. The second parameter includes the number of items to be removed.

    {% highlight js %}$('#element').pillbox('removeItems', index, count);{% endhighlight %}

    .pillbox('items')

    Returns an array of objects (one per item) containing the jQuery data() contents of the item which includes data-* attributes and a text property with the label's visible text.

    {% highlight js %}$('#element').pillbox('items');{% endhighlight %}

    .pillbox('itemCount')

    Returns an integer representing the current number of items

    {% highlight js %}$('#element').pillbox('itemCount');{% endhighlight %}

    .pillbox('removeBySelector')

    Removes an item based on a jQuery selector

    {% highlight js %}$('#element').pillbox('removeBySelector', 'li[data-value=buzz]');{% endhighlight %}

    .pillbox('removeByText')

    Removes an item with contained text matching that of the provided text string parameter

    {% highlight js %}$('#element').pillbox('removeByText', "Item 4");{% endhighlight %}

    .pillbox('removeByValue')

    Removes an item with value matching that of the provided value string parameter. The item must include a .data-value="{value}" attribute

    {% highlight js %}$('#element').pillbox('removeByValue', '1');{% endhighlight %}

    Options

    You can pass options via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-value="".

    Name type default description
    acceptKeyCodes array [13, 188] Key codes for keys that trigger an add pill event. Default keys include 13 (for enter) and 188 (for camma).
    edit boolian false Enables user-edited pills
    onAdd function undefined A callback function that executes when a pill is added. function(data,callback){} The data parameter contains an array of the added elements. The callback parameter function provides asynchronous support for the add functionality. To add items, run the callback and provide an array of items to add. Provide the unaltered list of items to add as data as the parameter for the callback function callback(data).
    onRemove function undefined Function that runs when a pill is removed. function(data,callback){} The data parameter contains an array of the elements being removed. The callback parameter function provides asynchronous support for the remove functionality. To remove items, run the callback and provide an array of items to be removed. Provide the unaltered list of items to add as data as the parameter for the callback function callback(data).
    onKeyDown function undefined Function that executes when a keydown event is triggered. function(event,data,callback){} The event parameter contains the event object. The data parameter contains an array of the elements being removed. The callback parameter function that provides asynchronous support for the type-ahead functionality. Run the callback function to use the type-ahead drop-down with values. Provided the callback with an array of items to display in the type-ahead dropdown. {% highlight js %} callback({data:[ { text: '', value:'' }] }); {% endhighlight %}

    Events

    Event Type Description
    clicked.fu.pillbox This event triggers when a pill is clicked. The event returns a jQuery data() object with information about the clicked pill.
    added.fu.pillbox This event triggers when a pill is added. The event returns a jQuery data() object with information about the added pill.
    removed.fu.pillbox This event triggers when a pill is removed. The event returns a jQuery data() object with information about the removed pill.
    edited.fu.pillbox This event triggers when a pill is edited. The event returns a jQuery data() object with information about the edited pill.
