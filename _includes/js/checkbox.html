Usage

Via data-attributes

The checkbox will automatically instantiate any checkboxes with data-initialize="checkbox". If you add the control markup after page load with data-initialize="checkbox", this control will initialize on mouseover.

Via JavaScript

Call the checkbox control via JavaScript:

{% highlight js %}$('#MyCheckbox').checkbox();{% endhighlight %}

Methods

$().checkbox();

Activates your content as a custom checkbox

.checkbox('check')

Checks the checkbox

{% highlight js %}$('#element').checkbox('check');{% endhighlight %}

.checkbox('destroy')

Removes all functionality, jQuery data, and the markup from the DOM. Returns string of control markup.

{% highlight js %}$('#element').checkbox('destroy');{% endhighlight %}

.checkbox('disable')

Disables the checkbox

{% highlight js %}$('#element').checkbox('disable');{% endhighlight %}

.checkbox('enable')

Enables the checkbox

{% highlight js %}$('#element').checkbox('enable');{% endhighlight %}

.checkbox('isChecked')

Returns true or false depending on whether the checkbox is checked

{% highlight js %}$('#element').checkbox('isChecked');{% endhighlight %}

.checkbox('toggle')

Toggles the checkbox between checked/unchecked states

{% highlight js %}$('#element').checkbox('toggle');{% endhighlight %}

.checkbox('uncheck')

Unchecks the checkbox

{% highlight js %}$('#element').checkbox('uncheck');{% endhighlight %}
