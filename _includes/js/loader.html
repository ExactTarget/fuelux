Starting frame

Use the data-frame attribute to set the initial animation frame (defaults to 1). Additionally, this attribute can be modified at any time to set the current frame.

{% highlight html %}
{% endhighlight %}

Controlling speed

Use the data-delay attribute to alter animation speed (defaults to 150).

{% highlight html %}
{% endhighlight %}

Animation range

Use the data-begin and data-end attributes to control at which frames the animation begins and ends, respectively. (Defaults are 1 and 8)

Usage

Via data-attributes

The loader will automatically instantiate any loader with data-initialize="loader".

Via JavaScript

Call the placard control via JavaScript:

{% highlight js %}$('#MyPlacard').placard();{% endhighlight %}

Via JavaScript

Call the placard control via JavaScript:

{% highlight js %}$('#MyLoader').loader();{% endhighlight %}

Markup

Simply place class="loader" and data-initialize="loader" on a div. Easy, huh?

{% endhighlight %}

Methods

$().loader();

Attaches loader to a 'loader' classed div

.loader('next');

Increments the loader animation to the next frame

{% highlight js %}$('#element').loader('next');{% endhighlight %}

.loader('pause');

Pauses the loader animation at the current frame.

{% highlight js %}$('#element').loader('pause');{% endhighlight %}

.loader('play');

Plays the loader animation, resuming it if paused.

{% highlight js %}$('#element').loader('play');{% endhighlight %}

.loader('prev');

Decrements the loader animation to the previous frame.

{% highlight js %}$('#element').loader('prev');{% endhighlight %}

.loader('reset');

Resets the loader animation to the beginning frame.

{% highlight js %}$('#element').loader('reset');{% endhighlight %}
